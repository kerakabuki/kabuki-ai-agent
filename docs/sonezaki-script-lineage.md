# 『曽根崎心中』台本系統と著作権ガイド

> 作成: 2026-07-07（原文＝角川ソフィア文庫『曾根崎心中 冥途の飛脚 心中天の網島 現代語訳付き』入手を機に調査）
> 目的: 気良歌舞伎の動画・note記事・演目ガイド等の公開コンテンツ制作で、**どのテキストを引用してよいか**を迷わず判断できるようにする。

## 結論（運用ルール）

| テキスト | 著作権 | 公開コンテンツでの利用 |
|---|---|---|
| 近松門左衛門の原文（元禄16年・1703初演の浄瑠璃詞章） | パブリックドメイン | **引用可**。ただし引用前にPD底本（角川文庫の原文側・国会図書館デジタル等）と照合すること |
| 宇野信夫 脚色版（1953〜、現行松竹上演の標準台本） | **保護期間中**（宇野信夫は1991年没。没後70年＝2061年まで） | **台詞・ト書きの引用不可**。あらすじ・場面構成に触れるのは可（アイデア・プロットは著作権の保護対象外）だが、宇野版由来の場面は出典を明示する |
| 角川文庫の現代語訳・訳注（諏訪春雄） | 保護期間中 | **引用不可**。現代語訳が必要なら自前で書く（＝けらのすけの「超訳」として書き下ろす） |
| 気良歌舞伎の上演台本docx（`D:\気良歌舞伎\台本\曽根崎心中\曽根崎心中.docx`） | 系統不明・**宇野版系統の疑い濃厚**（下記） | **台詞を動画字幕・記事・演目ガイドに転載しない**。音声処理の内部照合用途（`scripts/joruri_extract/`）はOK。浄瑠璃（〽）部分は近松原文と一致するなら原文として引用可——必ず原文と照合してから |

## 原文 vs 宇野版の相違点

| 項目 | 近松原文（1703） | 宇野信夫版（1953復活上演〜現行松竹） |
|---|---|---|
| 開幕 | 観音廻り（お初の三十三所巡り）から始まる | 観音廻りはカット。生玉社前から |
| 田舎の客 | 生玉社前でお初を連れ去る**名前のない**客 | 天満屋の場に**「田舎客儀兵衛」**という名前付きの滑稽役として登場（現行上演でも配役される定番） |
| 徳兵衛の伯父 | 語りの中に出るだけで**舞台には登場しない** | **平野屋久右衛門が天満屋に登場**。原作にない**「九平次の悪事露見」の場面**が加えられ、徳兵衛の無実が（二人の死出の後に）明らかになる——宇野版最大の改変でヒットの要因 |
| 天満屋の亭主 | 名前なし（「亭主」） | **「天満屋惣兵衛」**と命名 |
| 幕切れ | 道行「この世の名残り、夜も名残り」→天神の森の心中まで詞章が続く | 同じく天神の森の心中場まで演じる |

**判定の目安**: 「儀兵衛」「惣兵衛」という役名、久右衛門の舞台登場、悪事露見の場——このいずれかがあれば宇野版（またはその系統）とみてよい。

## 気良台本docxの系統判定：宇野版系統で確定（2026-07-07）

台本docx（「令和八年気良歌舞伎 曽根崎心中」台本(仮)）を直接確認して確定した。根拠:

- 役名欄に**田舎客儀兵衛・平野屋久右衛門・天満屋惣兵衛・幇間彦丸**など宇野版固有の命名がそのまま並ぶ
- 場割りが宇野版と同一: **生玉社境内の場 → 天満屋の場 → 道行・曽根崎の森の場**（観音廻りなし）
- 天満屋の場に**原作にない「久右衛門登場〜九平次の悪事露見」の場面**があり、幕切れの台詞「徳兵衛、死ぬなよ、かならず死んでは、ならぬぞよ」まで宇野版の型の通り
- 台詞・ト書きは現代語の歌舞伎台本体で、近松原文の直接翻刻ではない

したがって**docxの台詞・ト書きは公開コンテンツ（動画字幕・記事・演目ガイド・配信）に転載しない**。音声処理の内部照合用途（`scripts/joruri_extract/`）はOK。

**〽の浄瑠璃部分について**: docxの〽詞章（「〽此の世の名残り夜も名残り…」等）は近松原文由来だが、**字句に脚色上の改変がある**（例: 原文「かならず添ふとすがり寄り」→docx「必ずさうと縋りより」、原文「罪を許して下されかし」→docx「罪をお許し下さりませ」）。引用はdocxからではなく、**`research/genbun/sonezaki-shinju/` の原文翻刻から**行うこと。

**上演と公開の区別（実務メモ）**: 非営利・無料・出演者無報酬の上演は著作権法38条1項により許諾不要とされるが、**動画のネット公開（公衆送信）や台本・字幕の複製配布は38条の対象外**。宇野版台詞が乗った上演映像をYouTube等に公開するのはリスクがある。浄瑠璃（原文由来）のみを活かす方針（joruri_extractの浄瑠璃抽出）や、原文ベースの台本への差し替えが安全。

## 引用可能な原文詞章（代表例）

**全文の翻刻を `research/genbun/sonezaki-shinju/` に収録**（角川文庫の原文パートより。観音廻り等3箇所欠落あり、詳細は同READMEを参照）。公開前に必ずPD底本と字句を照合すること（かな遣い・句読は底本により揺れる）。

**道行の冒頭（最も有名な一節）**

> この世の名残り、夜も名残り。死にに行く身をたとふれば、あだしが原の道の霜。一足づつに消えて行く、夢の夢こそあはれなれ。あれ数ふれば暁の、七つの時が六つ鳴りて、残る一つが今生の、鐘の響きの聞き納め。寂滅為楽と響くなり。

**結びの一節**

> 誰が告ぐるとは曽根崎の、森の雫と散りにけり。あはれこの世の名残のみかは。未来成仏疑ひなき、恋の手本となりにけり。

**天満屋・縁の下の場のお初の詞**（「死ぬる覚悟が聞きたい」を含むくだりは近松原文由来。既存記事 `note-articles/sonezakisinju.md` での引用はこのまま使用可）
