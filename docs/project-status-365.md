## AIアシスタント「けらのすけ」

| 項目 | 内容 |
|------|------|
| メインAI | **Gemini 2.5 Flash**（Google AI Studio 無料枠、14 RPM制御） |
| フォールバックAI | Workers AI Llama 3.1 8B（Gemini不可時に自動切替、常時有効） |
| RAG（検索拡張生成） | 演目・用語・FAQ・公演情報の4ソースから文脈構築（8,000字上限、3-gram類義語検索） |
| Function Calling | 5ツール: 公演検索 / ニュース検索 / 用語検索 / 演目検索 / 団体情報取得 |
| 会話記憶 | KV保存、直近20メッセージ（10往復）、1時間TTL |
| 安全対策 | RAGデータ厳密モード（DATA外の知識補足を禁止）、150文字制限、temperature 0.1 |
| 応答キャッシュ | 正規化クエリでキャッシュ、6時間TTL |

### AI処理フロー

```
ユーザー入力
    │
    ├── [キャッシュヒット] → 即返却（6h TTL）
    │
    ├── [レート超過] → Workers AI Llama 3.1 8B（フォールバック）
    │
    └── [通常] → RAG構築(8000字) → Gemini 2.5 Flash
                     │
                     ├── テキスト応答 → 返却 + キャッシュ保存
                     └── Function Call → ツール実行 → 再度Gemini → 返却
```
