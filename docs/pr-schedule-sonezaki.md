## 訴求ポイント

KABUKI PLUS+ なら **公演情報も予習も1つのアプリで完結** する：

```
LIVE（公演情報）                    NAVI（演目ガイド）
┌─────────────────────┐           ┌─────────────────┐
│ 南座 3月公演         │           │ 曽根崎心中       │
│ 曽根崎心中物語       │  演目紹介  │ あらすじ         │
│ 日程・配役           │ ────────→ │ 幕構成・配役     │
│ [演目紹介ボタン] ──┘│           │ 見どころ         │
└─────────────────────┘           └─────────────────┘
         ↓ 観劇後
   RECO（観劇記録）
┌─────────────────┐
│ 感想・写真を記録 │
```

ユーザーが自然に 公演情報 → 演目ガイド と遷移できる。

## 365ポストでの投稿方法

kabuki-post-365の既存機能で南座『曽根崎心中物語』のPR投稿を作成・管理できる。

### 使う機能

| 機能 | 使い方 |
|------|--------|
| **Posts API** | 公演期間に合わせた日付でポストを手動作成 |
| **キャラクター** | `お初`（曽根崎心中、character_id指定） |
| **テーマ** | 既存の曜日テーマ（演目/役者/豆知識）をそのまま活用 |
| **CTA** | `B`（公演情報→ LIVE URL）または `A`（詳しく知る→ NAVI URL） |
| **AI生成** | `POST /api/v1/generate/single/:postId` でGeminiにテキスト生成 |
| **4プラットフォーム** | Instagram / X / Facebook / Pinterest 自動生成 |

### 投稿の作り方

1. **ポストを作成**（POST `/api/v1/posts`）
   ```json
   {
     "post_date": "2026-03-XX",
     "day_of_week": 1,
     "theme": "演目",
     "character_id": <お初のID>,
     "special_day": "南座『曽根崎心中物語』開幕",
     "cta_type": "B",
     "cta_url": "https://kabukiplus.com/kabuki/live",
     "status": "draft"
   }
   ```

2. **AIでテキスト生成**（POST `/api/v1/generate/single/:postId`）
   - Geminiが4プラットフォーム分のテキスト+ハッシュタグを自動生成
   - `special_day` に公演情報を入れることで、公演連動の文面になる

3. **確認・編集** → ステータスを `approved` → 投稿後に `posted`

### Phase別のポスト設定例

| Phase | 日付 | theme | special_day | cta_type |
|-------|------|-------|-------------|----------|
| 予告 | 開幕2週前 | 演目 | 南座『曽根崎心中物語』まもなく開幕 | B（公演情報） |
| 予告 | 開幕1週前 | 役者 | お初と徳兵衛 — 近松門左衛門の純愛 | A（詳しく知る） |
| 開幕日 | 開幕日 | 演目 | 本日開幕！南座『曽根崎心中物語』 | B（公演情報） |
| 公演中 | 開幕3日後 | 豆知識 | 曽根崎心中は実際の心中事件が題材 | A（詳しく知る） |
| 公演中 | 中盤 | 豆知識 | 道行（みちゆき）とは？ | A（詳しく知る） |
| 千穐楽前 | 千穐楽前 | 演目 | 南座『曽根崎心中物語』まもなく千穐楽 | B（公演情報） |
| 千穐楽前 | 公演後半 | 役者 | 観劇ログ機能のモニター募集中！ | C（歌舞伎を楽しむ） |
| 千穐楽後 | 千穐楽翌日 | 役者 | 観劇記録をRECOで残そう | C（歌舞伎を楽しむ） |

### 既存365カレンダーとの関係

- 365カレンダーが生成済みの日付に **上書き**（`INSERT OR REPLACE`）してPR投稿を差し込む
- または、既存ポストの `special_day` を更新して公演テーマを追加する
- 曜日テーマ（演目/役者/豆知識）は崩さず、そのテーマの中で曽根崎心中を取り上げる形が自然
