# 新聞×地芝居研究・第2例 — 秋田魁新報見出しデータベースにみる地芝居の足跡

生成: 2026-07-04 ／ 手法実証: Claude Fable 5
先行研究: 土谷桃子「岐阜の地芝居の足跡: 岐阜日日新聞」(2018) — 地方紙渉猟による地芝居史研究のほぼ唯一の例（当プロジェクトの空白分析で確認: 新聞資料を主史料とする研究は全国568件中16件）
データ: 秋田県立図書館 ADEAC「秋田魁新報記事見出し検索」(1900〜1999年、見出し283,858件、無料公開)
収集: `scripts/collect_sakigake_headlines.mjs`（本プロジェクト開発） → `sakigake_headlines.json`（11キーワード・ユニーク200件）

---

## 要旨

秋田魁新報の見出しデータベースを11キーワードで機械的に渉猟した結果、**約半日の作業で次が得られた**: ①全国94団体リストに存在しない消滅・未登録の地芝居団体等の痕跡5件以上、②森岳歌舞伎の約40年にわたる盛衰の新聞記録（研究論文が扱っていない期間を埋める）、③藩政期の歌舞伎興行許可文書への手がかり。土谷が岐阜日日新聞で示した「新聞は地芝居研究の一級史料」という命題が、第2県でも成立することを確認した。手作業で数ヶ月を要する渉猟が、デジタルアーカイブ×自動収集で1日に短縮できることも実証した。

---

## 1. 発見①: リスト外の団体・一座の痕跡

現行の全国地芝居団体リスト（94団体）に秋田県は大森歌舞伎・森岳歌舞伎の2つしかない。しかし見出しだけから、以下が浮かび上がった:

| 発見 | 根拠記事（見出し） | 含意 |
|---|---|---|
| **前郷歌舞伎**（由利・消滅） | 「秋田の芸能から 1・2 前郷歌舞伎芝居」(1969-02-03/04)、「碑の周辺 村芝居の碑 **絶えて久し前郷歌舞伎 神社境内に残る舞台小屋**」 | 解説連載2回分の内容＋**碑と舞台小屋が現存**する可能性。現地調査候補の筆頭 |
| **紅梅団**（村芝居一座・消滅） | 「記録 村芝居『紅梅団』始末記」 | 「始末記」= 解散の経緯を記した記事。一座の生死が新聞に残る好例 |
| **上荒井歌舞伎芝居**（明治25年） | 「〔明治期〕二十五年（上荒井歌舞伎芝居、角館の曳き山、おばこ節）」 | 明治中期の上演記録。空白分析で「明治期研究は全国5件」— その空白を埋める種 |
| **柳橋歌舞伎** | 「柳橋歌舞伎」（単独見出し） | 詳細不明。紙面確認が必要 |
| **やまびこ演友会**（角館・素人一座） | 「出羽路に三度笠が踊った 角館町『やまびこ演友会』」(1973-01-04) | 地芝居と大衆演劇の境界事例 |

ほかに「倉の芝居小屋」（明治末・年表記事）、「旧藩期の芝居小屋」（昭和6年回顧）、藩政期文書「**乍恐口上書…歌舞伎芝居御免の願書**」（津軽侯下国時の興行許可願）の手がかりも得られた。

**含意: 「94団体」は現存団体の下限値であり、消滅団体を含む歴史的実数はその数倍とみられる。** 新聞渉猟は消滅団体を検出できる事実上唯一の網である（土谷の岐阜での発見と同型）。

## 2. 発見②: 森岳歌舞伎の「40年史」が新聞だけで書ける

論文ヒット10件（空白分析で県内最多）の森岳歌舞伎だが、新聞見出しは研究が扱わない時期を連続的に埋める:

| 年 | 見出し（要点） |
|---|---|
| 1959 | 満月の夜に開演 森岳の村歌舞伎 **ヨシズ張り舞台**で大見え |
| 1960 | **110年の伝統**を継ぐ 山本村歌舞伎の保存へ 県文化財専門委 |
| 1965 | 森岳名物を永久に 歌舞伎 保存会つくり後継者を |
| 1967-68 | 中学生が受け継ぐ／中学生らが初公演 |
| 1970 | 森岳に歌舞伎劇場 町内の集会場にも利用 |
| 1983-84 | 伝統が風前の灯 後継者難、減る一方の舞手 |
| 1995 | 「民俗文化を継ぐ」特集2回 |
| （年代未詳） | 「森岳子ども歌舞伎」と森岳小学校の取組 |

1960年記事の「110年の伝統」は**起源＝1850年前後という当時の証言**であり、由来の一次証拠。ヨシズ張り舞台（1959）→保存会（1965）→中学生継承（1967）→専用劇場（1970）→危機（1983）→子ども歌舞伎への転回、という**保存運動の全生活史**が見出しレベルで再構成できる。大森歌舞伎も同様に1986（保存会養成）・1991（3年ぶり復活か）・1995（特集）を確認。

## 3. 発見③: 記事分布は「論文の偏り」と相似形

200件の内訳: **康楽館（小坂の明治期芝居小屋）関連が約7割**（復元運動1983〜開館1986〜常設公演・観光化の記事群）、中央歌舞伎・一般文化記事が約3割、地元の地芝居実演記事は約35件。1980年代以降、地芝居の新聞表象が「上演の報道」から「保存・観光の言説」へ収斂していく——これは空白分析で見たテーマ分布（保存・政策・観光48件が平成以降に急増）と新聞レベルでも一致する。**演じる文化から守る文化への転換が、学術と報道の両方で同時に起きている。**

## 4. 手法の評価 — 見出し検索の射程と限界

- 「素人芝居」「農村歌舞伎」「地狂言」は**0件**。実態が無いのではなく、**見出しにその語彙が使われない**（本文検索できる岐阜日日新聞DBとの最大の違い）。「村芝居」3件・「地芝居」2件も同様。→ 見出しDBでは**固有名詞（座名・地名・小屋名）と「歌舞伎」広域検索**が正しい戦略
- 総当たり「歌舞伎」でも122件と処理可能な規模。283,858件の見出しは索引・年表・雑誌記事索引を含み、戦前分の粒度は粗い（1940年代以前の日付付き記事は2件のみ。戦前の実態は年表型レコードと紙面マイクロで補う必要）
- それでも**半日で骨格が引けた**。土谷が手作業で行った渉猟の「初回スクリーニング」は完全に自動化可能

## 5. 次のアクション

1. **紙面確認の優先リスト**（県立図書館マイクロ/複写取寄せ）: ①「秋田の芸能から 前郷歌舞伎芝居」1969-02-03/04 ②「文化 田舎歌舞伎 上中下」1954-08-07〜10 ③「秋田と歌舞伎 地芝居の形成について」1973-10-02（佐々木清） ④「村芝居の碑」「紅梅団始末記」該当号 ⑤「110年の伝統」1960-06-15
2. **前郷の現地情報**: 碑と舞台小屋の現存確認（由利本荘市前郷・八幡神社か）
3. **横展開**: 収集器はADEAC共通仕様 — 機関IDを差し替えるだけで他県の見出し・目録DBに適用可能。次候補は岐阜県内ADEAC機関（土谷研究の追試・拡張）と長野県
